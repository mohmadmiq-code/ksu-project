KSU STAT102 - Chapter 2 Frontend (GitHub Pages)

1) Upload files (index.html, styles.css, app.js, ai-mark.svg) to your repo root.
2) In app.js, set API_URL to your Cloudflare Worker endpoint:
   https://<your-worker>.workers.dev/api/ask
3) Enable GitHub Pages: Settings -> Pages -> Deploy from branch (main / root)

Features:
- Book-locked Q&A for Chapter 2 (2.1 + 2.2)
- Upload image question (sent to worker as data URL)
- CSV uploader + interactive charts (Bar / Pie / Histogram / Dot)
